Основы Python

Python — это высокоуровневый язык программирования, который отличается простотой и читаемостью кода.
Он широко используется для разработки веб-приложений, анализа данных, искусственного интеллекта и многого другого.

Основные концепции

Переменные и типы данных

                                
name = "Alice" # строка
age = 30 # целое число
height = 5.7 # число с плавающей запятой
is_student = True # логическое значение

Условные операторы

                                
if age >= 18:
print("Взрослый")
else:
print("Несовершеннолетний")

Циклы

                                
# Цикл `for`
for i in range(5):
print(i)

# Цикл `while`
while count < 5:
print(count)
count += 1

Функции

return f"Hello, {name}!"

Python — интересные факты!

  • Создатель: Python был создан Гвидо ван Россумом и впервые выпущен в 1991 году.

    Гвидо ван Россум — голландский программист, прежде всего известный как автор языка программирования Python. Среди разработчиков Python Гвидо известен как «великодушный пожизненный диктатор» проекта, что означает, что он продолжает наблюдать за процессом разработки Python, принимая окончательные решения, когда это необходимо. С июля 2018 года Гвидо ушёл в постоянный отпуск от диктаторства, оставив за собой право быть обычным разработчиком.

  • Имя: Название языка происходит от британского комедийного шоу "Monty Python's Flying Circus".

    «Летающий цирк Монти Пайтона» — британский комедийный скетч-сериал комик-группы «Монти Пайтон». Шоу выделялось своими сюрреалистическими сюжетами, рискованными намёками, яркими гэгами и искромётными диалогами.

  • Читаемость: Код на Python часто сравнивают с английским языком, что делает его идеальным для начинающих. 

                                            
print(...) —> распечатать (вывести на экран | в терминал, объект ...)
                                        
                                    
                                            
input() —> ввод (получение пользовательского ввода)
                                        
                                    
                                            
for i in range(5): ... —> для переменной `i` в диапазоне [0; 5), выполнить ...
                                        
                                    
                                            
while count < 5: ... —> пока значение переменной `count` меньше 10, выполнить ...
                                        
                                    
                                            
break —> разрыв (используется в цикле для его прерывания, выхода)
                                        
                                    
                                            
try: ... —> попытка (попытка выполнения кода ...)
                                        
                                    
                                            
finally: ... —> в конце концов (используется в блоке try/except для финального действия ...)
